# 弱电机柜模拟安装

拖拽式机柜设备模拟安装工具，支持真实毫米标尺、设备自定义命名和颜色，导出高清 PNG 图片。

🔗 **在线使用：https://realsenses0371.github.io/rack-simulator/**

## 使用方式

浏览器直接打开 `index.html` 即可，无需安装。

## 功能

- 🖱️ 拖拽 1U/2U/3U/4U/6U/8U 设备到机柜
- 🏷️ 自定义设备名称和颜色（交换机/配线架/理线器/服务器/防火墙/路由器/UPS/显示器 等预设）
- 📏 真实毫米标尺：右轨显示每 U 距地面高度（1U=44.45mm）
- 🔄 设备上下移动调整位置，支持鼠标拖拽设备块
- 📐 24U(1255mm) / 32U(1610mm) / 42U(2055mm) 机柜高度切换
- 📸 导出高清 PNG 图片
- 📱 手机端适配：点击选中 + 点击放置，设置面板可折叠

## 机柜规格

| 高度 | 总高 | 安装区 |
|------|------|--------|
| 24U | 1255mm | 1067mm |
| 32U | 1610mm | 1422mm |
| 42U | 2055mm | 1867mm |

## 技术

纯 HTML + CSS + JavaScript，单文件，零依赖运行。html2canvas (CDN) 用于图片导出。
